# Sonata

An experiment with Skia/CanvasKit, using JSX to render graphics and provide
reactivity and user interaction.

## Getting Started

You'll need Yarn v1.x and a recent version of Node

```sh
yarn
yarn dev
```

Files worth looking at:

-   [./components/Surface.tsx]
-   [./renderer/engine.tsx]
-   [./renderer/Scene.tsx]
